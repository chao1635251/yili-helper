# 伊犁环线 iPhone 专用 PWA 安装版

## 最推荐安装方式
1. 将整个文件夹上传到极空间 NAS 的 Web 目录，或上传到 GitHub Pages / 个人网站。
2. 用 iPhone Safari 打开 `index.html` 对应的网址。
3. 点击底部分享按钮。
4. 选择“添加到主屏幕”。
5. 桌面出现“伊犁助手”图标后，打开一次，等待页面显示“已启用离线缓存”。
6. 之后无网络也可查看路线、打卡点、规则和检查清单。

## 注意
- PWA 离线缓存需要通过 http/https 访问，直接打开本地 html 文件时通常无法真正安装和离线缓存。
- iPhone 必须用 Safari 添加到主屏幕，微信/夸克/其他浏览器不稳定。
- 这个是路线助手，不是实时导航。实际行驶以高德/百度地图和新疆路况公告为准。
